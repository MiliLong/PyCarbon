# PyCarbon

一个简单的`python`爬虫脚本，可以将你的代码通过`https://carbon.now.sh`网站的`API`导出成`PNG`，`SVG`格式。

A simple `python` scraper script that can export your code to `PNG`, `SVG` format through the `API` of the `https://carbon.now.sh` website.
