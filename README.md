# PyCarbon

一个简单的`python`爬虫脚本，可以将你的代码通过`https://carbon.now.sh`网站的`API`导出成`PNG`，`SVG`格式。

## 官方主页

![CarbonPage](./img/CarbonPage.png)

## Example

```python
print('hello world')
```
